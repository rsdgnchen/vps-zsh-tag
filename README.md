# Vps-Zsh-TagBoard 🏷️

Debian 延伸版专用 Zsh 看板脚本，为你的 VPS 终端提供优雅的信息展示和现代化的 Zsh 体验。

## ✨ 功能特性

### 🎨 智能终端美化
- 自动安装并配置 Zsh Shell
- 根据操作系统自动着色（Debian/Ubuntu/Kali/Raspbian/Pop!_OS/Deepin/Devuan 等）
- 增强的命令补全功能（菜单式选择）
- 智能历史记录搜索（上下箭头快速匹配）
- 5000 条历史记录共享

### 🏷️ 自定义看板系统
- **三行自定义标签**：厂商/用途、位置/归属、网络/警示
- **灵活的到期管理**：
  - 支持月付/季付/半年/年付周期
  - 自动计算到期日期和剩余天数
  - 7 天内到期红色警告
  - 永久标识模式（∞）
- **系统信息展示**：CPU 核心数、内存大小、操作系统版本

### 🔧 管理功能
- **一键安装**：完整配置 Zsh + 看板
- **安全移除**：仅移除看板，保留 Zsh 配置
- **彻底卸载**：还原 Bash、清除所有配置、可选移除 Zsh 包
- **自动备份**：安装前自动备份原有 `.zshrc`
- **权限安全**：禁止对 root 用户配置，自动设置 sudo 免密

```bash

```

## 📖 使用方法

### 主菜单
运行脚本后，提供三个选项：

```
===================================
    看板管理脚本
===================================
  1) 安装看板
  2) 移除看板
  3) 彻底卸载
===================================
```

### 选项说明

| 选项 | 功能 | 影响范围 |
|------|------|---------|
| **1) 安装看板** | 完整安装 Zsh + 看板系统 | 安装 Zsh，配置 Shell，创建看板 |
| **2) 移除看板** | 仅删除看板文件 | 保留 Zsh 和 .zshrc 配置 |
| **3) 彻底卸载** | 还原所有更改 | 还原 Bash，清除配置，可选移除 Zsh |

### 安装流程
1. 选择目标用户（**禁止 root**，可自动创建新用户）
2. 配置 sudo 免密权限
3. 设置默认 Shell 为 Zsh
4. 输入三行自定义标签信息
5. 配置到期规则（可选永久模式）
6. 自动生成看板并应用配置

### 备份说明
- 安装前自动备份 `.zshrc` 为 `.zshrc.YYYYMMDD.bak`
- 备份文件保存在用户主目录
- 卸载时需手动删除 `.zshrc`（如不再使用 Zsh）

## 🔒 安全特性

- **禁止 root 配置**：防止误操作影响系统安全
- **权限隔离**：所有文件归属于目标用户
- **输入验证**：严格的日期和参数校验
- **sudoers 保护**：正确设置权限 0440

## 📁 生成的文件

安装后会在用户主目录生成以下文件：

| 文件 | 用途 |
|------|------|
| `~/.zshrc` | Zsh 主配置文件（含自动备份） |
| `~/.zprofile` | Zsh 登录配置（自动加载看板） |
| `~/.welcome.sh` | 看板脚本（展示系统信息和标签） |
| `~/.welcome.conf` | 看板配置文件（存储用户数据） |
| `/etc/sudoers.d/<用户名>` | sudo 免密配置 |

## 🛠️ 系统要求

- **操作系统**：Debian 及其衍生版（Ubuntu、Kali、Raspbian 等）
- **权限要求**：root 或 sudo 权限运行
- **网络要求**：首次安装需下载 Zsh 包
